# CLAUDE.md

## What This Is

Healthcare tenant lead generation pipeline. Finds healthcare businesses in Asheville, NC via Google Places API, enriches with drive times and multi-location detection, exports ranked leads to CSV/Excel. The end user is the property owner (user's father).

## How to Run

```bash
source .venv/bin/activate
python collect.py    # Google Places API data collection
python enrich.py     # Drive times, org detection, website scraping
python export.py     # CSV + Excel export to data/
```

## Key Files

- `config.py` — search categories, property coordinates, API endpoints, constants
- `db.py` — SQLite schema and CRUD helpers
- `collect.py` — Google Places Text Search collection
- `enrich.py` — Routes API drive times, multi-location detection, email/description scraping
- `export.py` — ranked CSV + formatted Excel export
- `data/leads.db` — SQLite database (gitignored, generated by pipeline)

## Rules

- `.env` contains API keys — never commit it. It's in `.gitignore`.
- `data/` is gitignored. All generated output goes there.
- Don't create new files without good reason. The pipeline is 5 Python files — keep it that way.
- The plan file has the full implementation spec. Check it before making architectural changes.
