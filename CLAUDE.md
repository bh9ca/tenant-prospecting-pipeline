# CLAUDE.md

## What This Is

Healthcare tenant lead generation pipeline for two retail properties in Arden, NC. Finds healthcare businesses via Google Places API, enriches with drive times and multi-location detection, classifies by business type, scores leads into tiers (A/B/C/D), and exports ranked leads to CSV/Excel. The end user is the property owner (user's father).

### Properties

- **Shops on Rockwood (330-336 Rockwood Rd)**: 34,405 SF GLA, 12 stores, built 2006. 2 spaces available: 2,200-3,300 SF ($19/SF/YR) + 1,500 SF ($24/SF/YR). Total available: 4,800 SF.
- **Shops at Airport Road (316 Rockwood Rd)**: 9,000 SF freestanding, under construction (2026). 1 space: 1,500-9,000 SF ($26/SF/YR Net).
- **Tenant sweet spot: 1,500-9,000 SF**

## How to Run

```bash
source .venv/bin/activate
python collect.py    # Google Places API data collection
python enrich.py     # Drive times, classification, org detection
python export.py     # CSV + Excel export to data/
```

Pipeline is re-runnable: `collect.py` skips already-searched queries, `enrich.py` skips existing drive times/emails, org detection re-runs from scratch.

## Key Files

- `config.py` — search categories, property coordinates, API endpoints, type mappings, tier definitions
- `db.py` — SQLite schema, CRUD helpers, migration logic
- `collect.py` — Google Places Text Search collection
- `enrich.py` — Routes API drive times, business type classification, org detection with address normalization
- `export.py` — tier scoring, ranked CSV + Excel export
- `data/leads.db` — SQLite database (gitignored, generated by pipeline)

## Organization Detection & Address Dedup

Businesses are grouped into organizations by website domain, then distinct physical locations are counted via address normalization within each org:

1. **Same address, DIFFERENT domains** → separate orgs (independent practices sharing a building). No dedup. Correct by design.
2. **Same address, SAME domain** → same org, one location. Dedup via address normalization (strips suite/unit/floor).
3. **Different addresses, SAME domain** → same org, multiple locations.

This means a medical office building with 5 independent practices (each with their own website) = 5 separate leads. That's correct.

## Output

CSV is the primary format (opens natively in Google Sheets). End goal is a dashboard.

- `data/leads_ranked.csv` — all businesses with tier + category
- `data/top_prospects.csv` — Tier A + B only (actionable list)
- `data/organizations.csv` — multi-location orgs
- `data/leads.xlsx` — Excel workbook with same data
